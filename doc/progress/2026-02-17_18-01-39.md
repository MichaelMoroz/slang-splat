# Raster Backward Sync Cleanup

## Change
- Removed an unnecessary GroupMemoryBarrierWithGroupSync() between:
  - prefetch_batch_from_global(...)
  - clear_batch_gradient_cache(...)

## Rationale
- Prefetch writes groupshared gaussian cache arrays.
- Gradient clear writes groupshared gradient-state cache.
- These writes target disjoint groupshared memory, so an intermediate sync is unnecessary.
- A single sync after both operations still guarantees visibility before the backward loop consumes shared data.
