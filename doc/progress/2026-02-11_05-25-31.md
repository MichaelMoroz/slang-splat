# Remove Unused Atomic Output Values

## Summary
- Removed unused `old` outputs from gradient `InterlockedAdd` calls.
- Added a scalar shared helper `atomic_add_grad_shared` and reused it from `atomic_add_grad4_shared`.
- Kept the shared-cache + global-flush gradient algorithm unchanged.

## Changes
- `shaders/renderer/gaussian_raster_stage.slang`
  - `atomic_add_grad<...>` now calls `InterlockedAdd(..., value)` directly.
  - Added `atomic_add_grad_shared(uint idx, float value)` for shared-cache atomics.
  - Refactored `atomic_add_grad4_shared(...)` to use the scalar helper in the unrolled loop.
