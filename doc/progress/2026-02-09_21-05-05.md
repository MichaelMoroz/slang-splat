## Remove Redundant Screen Valid Buffer

Removed `g_ScreenValid` from the GPU renderer pipeline because rasterization only consumes splats emitted by binning/sorting, which are already visibility-filtered.

Changes:

- `shaders/renderer/gaussian_pipeline.slang`
  - Removed `g_ScreenValid` uniform and all writes in `csProjectAndBin`.
  - Removed `gsValid` groupshared cache and inner-loop validity check in `csRasterize`.
- `src/renderer/gaussian_renderer.py`
  - Removed `screen_valid` work buffer allocation.
  - Removed `g_ScreenValid` kernel bindings for project and raster passes.
  - Removed `screen_valid` from `debug_pipeline_data()` readback payload.
