# MVEE normalization + legacy orientation correction

## Changes
- Shader MVEE fit now normalizes input points before solving:
  - subtract mean center,
  - divide by max radial extent,
  - solve in normalized space,
  - map center/radius/conic back to screen space.
- Legacy anisotropic axis projection now uses inverse quaternion (`qInv`) for local->world axis rotation, fixing orientation mismatch.
- CPU reference updated to match:
  - silhouette ONB generation now uses the same stable branchless basis construction,
  - MVEE fit now uses the same point normalization/de-normalization strategy.
