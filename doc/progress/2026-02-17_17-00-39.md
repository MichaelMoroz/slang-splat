# Ellipsoid-cache backprop refactor + raster binding fix

## Summary
- Removed global g_Splat* intermediate buffers from projection/raster/training paths.
- Kept ellipsoid-space data only in groupshared cache for raster forward/backward.
- Converted cached ellipsoid gradients to base Gaussian parameter gradients during batch flush.
- Simplified fused ADAM kernel to consume base-parameter gradients directly.
- Fixed raster scene rendering regression by binding base scene buffers in raster/raster-backward dispatch.
- Updated test coverage and gradient API names.

## Shader changes
- shaders/renderer/gaussian_types.slang
  - Removed g_SplatPosLocal/g_SplatInvScale/g_SplatQuat uniforms.
  - Replaced old grad buffers with g_GradPositions/g_GradScales/g_GradRotations/g_GradColorAlpha.
  - Marked Gaussian3D and Ellipsoid as differentiable structs for AD path.
- shaders/renderer/gaussian_project_stage.slang
  - Removed writes to deleted g_Splat* buffers.
- shaders/renderer/gaussian_math_utils.slang
  - Marked uild_ellipsoid as differentiable.
- shaders/renderer/gaussian_raster_stage.slang
  - Added differentiable cached-ellipsoid builder from base params.
  - Prefetch now builds ellipsoid from base scene buffers and stores only cached derived values in groupshared.
  - Backward accumulates cached derived grads in groupshared, then converts to base grads in lush_batch_gradient_cache via AD helper backward.
  - Clears/writes only new base gradient buffers.
- shaders/renderer/gaussian_training_stage.slang
  - Removed splat-space gradient inputs and manual local->world chain logic.
  - csAdamStepFused now reads direct base gradients.
  - Kept numeric guards/clamps/normalization policy.

## Python runtime changes
- src/renderer/gaussian_renderer.py
  - Removed splat intermediate and old grad work buffers.
  - Added base grad work buffers (grad_positions, grad_scales, grad_rotations, grad_color_alpha).
  - Updated raster clear/backward bindings to new grad buffers.
  - Added missing base scene buffer bindings in raster/raster-backward dispatch to restore visible rendering.
  - Updated debug gradient readback keys.
- src/training/gaussian_trainer.py
  - Updated ADAM dispatch bindings to new base grad buffers.
  - Removed obsolete training uniforms from common vars (g_RadiusScale, g_CamPos, minInvScale upload).
