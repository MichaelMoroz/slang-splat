# UI projection-mode toggle in realtime viewer

## What changed
- Added a `Projection Mode` combo box to the viewer `Render Params` panel.
- Exposed two runtime options:
  - `Legacy Axis Extent` (`legacy_axis_extent`)
  - `Sampled5 MVEE` (`sampled5_mvee`)
- Selecting a new mode now updates `self.projection_mode` and recreates `GaussianRenderer` with the same window size and current slider settings.
- Existing scene binding is preserved after renderer recreation.

## Why
- Enables live A/B switching between projection/binning paths from the UI without restarting the app.
- Keeps CLI `--projection-mode` as the initial mode while allowing runtime overrides.
