# Progress 2026-02-18 03:55:42

## Implemented
- Added threshold-based low-quality splat reinitialization pipeline in `shaders/renderer/gaussian_training_stage.slang`.
  - New uniform: `g_LowQualityReinitEnabled`.
  - New buffer: `g_LowQualityFlags`.
  - New helper: `is_low_quality(scale, opacity)` with inclusive threshold checks.
  - New kernels:
    - `csMarkLowQualitySplats`
    - `csResampleLowQualitySplatsRandom`
- `csResampleLowQualitySplatsRandom` behavior:
  - Runs per low-quality splat.
  - Samples deterministic random donor from `(splatId, g_Adam.stepIndex)`.
  - Skips when donor is low-quality.
  - Copies donor params, applies optional position jitter using `g_MCMC.positionNoiseScale`, reclamps/sanitizes, and resets ADAM moments for replaced splats.
