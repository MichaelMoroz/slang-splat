## Group-Wide Raster Early-Out

Updated `shaders/renderer/gaussian_pipeline.slang` to stop `csRasterize` batch processing when all pixels in a threadgroup have reached `pixelDone`.

Algorithm:

- Added `groupshared uint gsStoppedPixels`.
- Before the batch loop:
  - lane 0 clears `gsStoppedPixels`,
  - group synchronizes,
  - lanes already done (`pixelDone == true`) add to `gsStoppedPixels`.
- During the batch loop:
  - splat data load uses the existing post-load barrier,
  - lanes that become done due to transmittance threshold atomically add to `gsStoppedPixels`,
  - existing end-of-batch barrier synchronizes atomics,
  - break when `gsStoppedPixels == RASTER_BATCH`.

This preserves uniform barrier participation while enabling group-level early termination.
