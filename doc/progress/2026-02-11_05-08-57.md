# Raise default prepass cap to 4 GB

## Summary
- Increased the default prepass memory cap from `512 MB` to `4096 MB`.
- This keeps the cap safety mechanism but reduces overflow/cap-limited behavior for large scenes by default.

## Changes
- `src/renderer/gaussian_renderer.py`
  - `GaussianRenderer(..., max_prepass_memory_mb=4096)` default updated.
- `viewer.py`
  - `SplatViewer(..., max_prepass_memory_mb=4096)` default updated.
  - CLI `--prepass-memory-mb` default updated to `4096`.
- `render.py`
  - CLI `--prepass-memory-mb` default updated to `4096`.
