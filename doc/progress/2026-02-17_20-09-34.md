# 2026-02-17 20:09:34 - MCMC position perturbation with Box-Muller and hash RNG

## Summary
Implemented MCMC-style random position perturbation in the fused ADAM kernel, with Gaussian noise sampled via Box-Muller and anisotropy driven by each Gaussian's scale+rotation. Replaced per-Gaussian RNG state buffer with stateless hash RNG seeded by gaussian id and optimizer step id.

## Shader changes
- `shaders/renderer/gaussian_training_stage.slang`
  - Added `TrainingMCMCParams` uniform block (`enabled`, `positionNoiseScale`, `opacityGateSharpness`, `opacityGateCenter`).
  - Added RNG utilities:
    - LCG step
    - `hash_u32` seed mixer
    - Box-Muller normal sampling (`rng_normal3_box_muller`)
  - Added position-only perturbation in `csAdamStepFused`:
    - opacity-gated magnitude via `sigmoid(-k(opacity - t))`
    - anisotropic noise as rotated scaled normal vector (`R * (scale * eta)`)
    - scaled by position LR and user noise scale
  - Removed `g_RngState` buffer dependency; RNG now uses hashed `(splatId, stepIndex)` seed.

## Runtime changes
- `src/training/gaussian_trainer.py`
  - Extended `TrainingHyperParams` with MCMC controls:
    - `mcmc_position_noise_enabled`
    - `mcmc_position_noise_scale`
    - `mcmc_opacity_gate_sharpness`
    - `mcmc_opacity_gate_center`
  - Bound `g_MCMC` uniforms in `_common_vars`.
  - Removed `rng_state` buffer allocation/upload/binding.

## UI changes
- `viewer.py`
  - Added optimizer controls:
    - `MCMC Pos Noise` (checkbox)
    - `MCMC Noise Scale`
    - `MCMC Opacity K`
    - `MCMC Opacity T`
  - Wired controls into `TrainingHyperParams` collection.
