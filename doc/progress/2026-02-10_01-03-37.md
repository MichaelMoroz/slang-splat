# Merge tiny renderer shader modules

## Changes
- Removed undersized renderer shader files by merging their contents into larger modules:
  - `gaussian_bindings.slang` -> merged into `gaussian_types.slang`
  - `gaussian_tile_range_stage.slang` -> merged into `gaussian_project_stage.slang`
  - `gaussian_pipeline.slang` removed; entrypoint wrappers now live in `gaussian_project_stage.slang`
- Updated imports across renderer shaders to use `gaussian_types.slang` as the shared core module for constants, bindings, groupshared state, and structs.
- Updated renderer shader load path in `src/renderer/gaussian_renderer.py`:
  - from `gaussian_pipeline.slang`
  - to `gaussian_project_stage.slang`
