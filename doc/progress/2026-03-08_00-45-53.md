# Refactor Continuation And Dead-Code Removal

## Scope
- Continued the whole-repo Python complexity refactor with another cleanup pass over viewer, training, renderer, and scene package surfaces.
- Removed wrapper/package files that were no longer needed after the earlier package split.

## Code Changes
- Deleted unused package adapters: `src/app/__init__.py`, `src/viewer/__init__.py`, `tools/__init__.py`.
- Deleted the unused COLMAP compatibility module `src/scene/colmap_loader.py` and kept the public `src.scene` API by importing the same symbols directly from the split internal modules in `src/scene/__init__.py`.
- Removed the unused `apply_scene_camera_fit()` helper from `src/app/shared.py` and updated imports to use `src.app.shared` directly where needed.
- Simplified `src/viewer/session.py` by removing low-value wrappers, consolidating COLMAP point-table extraction, centralizing renderer disposal/sync invalidation, and keeping renderer recreation plus training-scene sync paths shorter.
- Simplified `src/viewer/app.py` and `src/viewer/presenter.py` to consume the shared camera-fit logic and inline a few trivial selection wrappers that only added function count.
- Simplified `src/training/gaussian_trainer.py` by collapsing repeated renderer scene/gradient buffer bindings behind mapping helpers, deleting unused helper APIs, and folding point-buffer packing/image conversion into the actual upload/texture creation paths.
- Simplified `src/renderer/gaussian_renderer.py` by deleting an unused buffer-usage helper and reusing shared helpers for scene/screen/grad shader var assembly plus render-stat/background normalization.
