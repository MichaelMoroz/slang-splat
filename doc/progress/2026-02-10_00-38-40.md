# Index validation hardening for scanline binning

## Changes
- Added strict finite/range checks in `compute_scanline_tile_span` (`gaussian_pipeline.slang`) before float-to-int conversion.
- Added final safety checks in `emit_splat_bins` write pass:
  - validate `lineMinX`/`tileY` against tile dimensions
  - validate `writeIndex` against `g_maxListEntries`
  - validate `tileId` against `g_tileCount`
- Fixed uniform binding bug in `GaussianRenderer._project_and_bin` by passing `g_tileCount` so shader-side tile-id validation works correctly.
