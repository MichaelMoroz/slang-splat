# Raster Stage OOP/Generic Refactor

## Summary
- Replaced duplicated raster gradient atomic helpers with a generic accumulation path.
- Moved raster state update logic into `RasterState` methods.
- Introduced `RasterGaussian` with method-based global/shared loads and method-scoped custom backward derivatives.

## Code Changes
- `shaders/renderer/gaussian_raster_stage.slang`
  - Added generic `atomic_add_grad<let N : int>(...)` and unified `atomic_add_grad4(...)`.
  - Added `RasterState.blend(...)` and `RasterState.undo(...)` methods and routed blend/undo logic through them.
  - Added `RasterGaussian` struct with:
    - `load_global(...)` and `load_shared(...)` static methods.
    - custom derivative methods `raster_gaussian_load_global_bwd(...)` and `raster_gaussian_load_shared_bwd(...)`.
    - `store_shared(...)` method for batch prefetch.
  - Updated `raster_eval_alpha`, `raster_blend_step`, and prefetch logic to use `RasterGaussian` methods.
