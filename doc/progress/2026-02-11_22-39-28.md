# Shader Uniform Struct Grouping

## Summary
- Replaced the large flat shader uniform parameter set with grouped struct uniforms.
- Migrated camera, prepass, and raster parameters to `g_Camera`, `g_Prepass`, and `g_Raster`.
- Updated Python-side parameter binding and indirect cursor setup to populate struct fields.

## Changes
- `shaders/renderer/gaussian_types.slang`
  - Added `CameraParams`, `PrepassParams`, and `RasterParams`.
  - Replaced flat scalar uniforms with:
    - `uniform CameraParams g_Camera;`
    - `uniform PrepassParams g_Prepass;`
    - `uniform RasterParams g_Raster;`

- `shaders/renderer/gaussian_math_utils.slang`
  - Switched all camera/projection/depth/radius uniform reads to struct fields (`g_Camera.*`, `g_Prepass.*`).

- `shaders/renderer/gaussian_binning_utils.slang`
  - Switched visibility and tile/capacity reads to struct fields.

- `shaders/renderer/gaussian_scanline_utils.slang`
  - Switched loop limits to `g_Prepass.tileWidth` / `g_Prepass.tileHeight`.

- `shaders/renderer/gaussian_project_stage.slang`
  - Migrated prepass, camera, and depth-pack reads to `g_Prepass.*` and `g_Camera.*`.

- `shaders/renderer/gaussian_raster_stage.slang`
  - Migrated raster controls and debug flags/colors to `g_Raster.*`.
  - Migrated shared tile/splat settings to `g_Prepass.*`.
  - Migrated camera ray setup to `g_Camera.*`.

- `src/renderer/camera.py`
  - `gpu_params(...)` now returns camera struct fields (`viewport`, `camPos`, `camBasis`, `focalPixels`, `nearDepth`, `farDepth`) for `g_Camera`.

- `src/renderer/gaussian_renderer.py`
  - Added struct-binding helpers:
    - `_camera_uniforms(...)`
    - `_prepass_uniforms(...)`
    - `_raster_uniforms(...)`
    - `_bind_prepass_cursor(...)`
  - Updated all dispatches/cursors to bind `g_Camera`, `g_Prepass`, and `g_Raster`.

- `doc/Rendering.md`
  - Added a new section documenting uniform struct layout and binding model.
